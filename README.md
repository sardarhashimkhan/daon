# DaonFrameworks

DaonFrameworks is a collection of iOS SDKs for biometric authentication and document verification.  
This package includes:
 
- **Daon Face SDK**  
- **Daon FIDO SDK**  
- **Daon Document SDK**  
- **Dependant Frameworks**
